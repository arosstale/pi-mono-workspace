## What's New in V2.5

### 1. ALMA Meta-Learning Agent ğŸ†•

Based on the ALMA research paper (https://arxiv.org/pdf/2602.07755):

**Key Innovation**: AI systems should learn **HOW to optimize**, not just **WHAT to execute**.

#### Features

- **Design Proposal**: Mutates existing memory designs or creates random ones
- **Evaluation**: Measures performance (accuracy, efficiency, compression)
- **Archive**: Stores designs with scores
- **Iteration**: Continuously improves through mutations
- **Best Design Tracking**: Always uses the highest-scoring design

#### Advanced Capabilities (New in Enhanced V2.5)

- **Real Evaluator**: Uses actual memory performance metrics instead of mocks.
- **Advanced Mutations**: Gaussian, Simulated Annealing, Crossover, and Adaptive strategies.
- **V7 Integration**: Dedicated script (`alma_v7_integration.py`) to apply ALMA to trading strategies.

### 2. Knowledge System (New) ğŸ†•

A semantic search engine for the agent's own codebase.

- **Indexer**: Indexes code files for retrieval.
- **Searcher**: Finds relevant code snippets based on queries.
- **Registry**: Central component management system.

### 3. ALMA+PAOM Integration ğŸ†•

**Combines**:
- **ALMA**: Discovers optimal memory designs
- **PAOM**: Implements memory design
