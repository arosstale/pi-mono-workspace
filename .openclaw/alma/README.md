## What is ALMA?

ALMA is a meta-learning system that automatically discovers optimal memory designs
for agentic AI through continuous evaluation and mutation.

### Key Features

- Dynamic consensus weights based on performance
- Market regime detection
- A/B testing for design comparison
- Performance tracking with rollback safety
